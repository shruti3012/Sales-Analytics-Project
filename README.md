# Sales Analytics ETL Project

## Project Overview
This project demonstrates an end-to-end ETL pipeline using Python and SQL.

## Technologies Used
- Python
- Pandas
- DuckDB
- CSV
- Parquet

## Features
- Read raw CSV files
- Data validation
- Join multiple datasets
- Calculate TotalSales
- Generate analytics reports
- Save processed data
- Export to Parquet
